# Personal Portfolio

My personal portfolio website built with React, TypeScript, and Vite. Features a modern, responsive design with smooth animations and accessibility in mind.

## 🚀 Features

- Modern, responsive design
- Smooth animations using Framer Motion
- Accessible and SEO-friendly
- Dark theme support
- Social media integration
- Optimized performance

## 🛠️ Built With

- React
- TypeScript
- Vite
- Emotion (Styled Components)
- Framer Motion
- Font Awesome

## 📥 Installation

1. Clone the repository:
```bash
git clone https://github.com/huseyinrashid/portfolio.git
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

## 🚀 Deployment

The site is automatically deployed to GitHub Pages when changes are pushed to the main branch.

To deploy manually:

1. Build the project:
```bash
npm run build
```

2. Push changes to the main branch:
```bash
git push origin main
```
